# Flight scraper

## Getting started
```bash
pipenv run python flight-selenium.py
```

## Prepare
### Pip
```bash
pip install pipenv
```
### Pipenv
```
pipenv update
```
### Chromedriver
Go at [Chromedriver Website](https://chromedriver.chromium.org/) and download the appropriate chromedirver.
Unzip the `chromedriver.exe` in this directory.
